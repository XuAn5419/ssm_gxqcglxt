## 项目说明

本网站的客户端和管理端使用h-ui框架来实现，服务端使用 **Spring MVC+ MyBatis** 来实现，数据库使用了 **MySQL**。

<br/>

## 项目功能

- 汽车租赁
- 用户登录注册
- 用户信息编辑、头像修改
- 车辆搜索
- 车辆评论，留言
- 后台对用户、车辆、订单信息的管理

<br/>

## 技术栈

### 后端

**SpringBoot + MyBatis**

### 前端

**H-ui+HTML+CSS+JSP**

<br/>

## 开发环境

JDK： jdk1.8
